The file "ONLY_SOSCI_discounting_procedure.html" is only meant to be run in the soscisurvey.de embeded HTML IDE. 
Running it in any other HTML environment wont work
For correct implemantation into SoSci, the correctly specified Instrumental Variables, saving the Subjective Valuations have to be created. Refer to the SoSci documentation. 

However, the file "discounting_procedure_runable.html" is runable. 

If executed it will not only display the Discounting Task from the Quetionnaire, it will also yield a csv-file with the provided valuations after comletion. 
The file will be named "demo_result.csv".
Put it into the "data_and_analysis" folder to perform the demo analysis.
